# smartplan-meteo

Application de planification intelligente avec météo pour équipes terrain.

## But
SmartPlan Météo est une application web qui aide à planifier des jobs terrain selon :
- la météo
- la priorité
- la durée
- le lieu
- les notifications client

## Fichier principal
- `index.html`

## Fonctionnalités déjà présentes
- dashboard
- calendrier jour / semaine / mois
- création de jobs
- planification automatique
- import/export `.ics`
- Google Calendar
- report de job
- préparation de SMS
- base de parsing vocal
- météo simulée
- drag and drop dans le calendrier
